# MindTrack
Academic stress insights using data science (EDA + ML) 
## Overview
MindTrack is a data science project that explores patterns in academic stress among university students.
The goal is to analyze how factors such as sleep, workload, screen time, and study habits relate to stress levels.
## Objectives
Analyze relationships between academic and lifestyle factors
Visualize stress-related trends using data
Build a simple machine learning model to identify high stress risk
## Data
The dataset used in this project is simulated student survey data.
It is created for educational and analytical purposes only.
## Tools & Technologies
Python
Pandas & NumPy
Matplotlib & Seaborn
Scikit-learn
## Disclaimer
This project is not a medical or diagnostic tool.
All data is synthetic and intended only for exploratory analysis.
